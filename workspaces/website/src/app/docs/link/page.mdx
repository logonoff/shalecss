import { Link } from "@shalecss/react";

The `Link` component is a polymorphic wrapper for creating styled links with state variant support.

## Props

*extends HTML anchor attributes*

- `state` (optional): The initial state of the link. Can be one of the following values:
  - `default`: The default state.
  - `hover`: The hover state.
  - `active`: The active state.
  - `disabled`: The disabled state.
- `Component` (optional): The underlying component to render. Defaults to `a`. This makes the component polymorphic.

## Example

### Basic Link

<Link href="/docs">Documentation</Link>

```tsx
import { Link } from "@shalecss/react";

const MyComponent = () => {
  return <Link href="/docs">Documentation</Link>;
};
```

### With State

```tsx
<Link href="/home" state="active">
  Home
</Link>
```

## Polymorphic Usage

Since Link is polymorphic, you can render it as any component. This is particularly useful for integrating with routing libraries:

```tsx
import { Link } from "@shalecss/react";
import NextLink from "next/link";

const MyComponent = () => {
  return (
    <Link Component={NextLink} href="/about">
      About
    </Link>
  );
};
```
