import { Icon } from "@shalecss/react";

The `Icon` component displays SVG icons from an icon sprite sheet using SVG's `<use>` element.

## Props

*extends `React.SVGAttributes<SVGSVGElement>`*

- `icon` (required): The icon ID from the icons.svg sprite file.
- `flipX` (optional): Whether to flip the icon horizontally. Defaults to `false`.
- `flipY` (optional): Whether to flip the icon vertically. Defaults to `false`.
- `iconPath` (optional): The path to the icons.svg file. Defaults to `/icons.svg`.

## Example

### Basic Icon

<Icon icon="home" />

```tsx
import { Icon } from "@shalecss/react";

const MyComponent = () => {
  return <Icon icon="home" />;
};
```

### With Custom Size

```tsx
<Icon icon="settings" width={24} height={24} />
```

### Flipped Icons

```tsx
{/* Flip horizontally */}
<Icon icon="arrow" flipX />

{/* Flip vertically */}
<Icon icon="arrow" flipY />

{/* Flip both directions */}
<Icon icon="arrow" flipX flipY />
```

### Custom Icon Path

If your icon sprite is located elsewhere:

```tsx
<Icon icon="logo" iconPath="/assets/icons.svg" />
```

## Icon Sprite Setup

The Icon component expects an SVG sprite file. Here's how to create one:

```xml
<!-- public/icons.svg -->
<svg xmlns="http://www.w3.org/2000/svg" style="display: none;">
  <symbol id="icon-home" viewBox="0 0 24 24">
    <path d="M10 20v-6h4v6h5v-8h3L12 3 2 12h3v8z"/>
  </symbol>

  <symbol id="icon-settings" viewBox="0 0 24 24">
    <path d="M19.14 12.94c.04-.3.06-.61.06-.94 0-.32-.02-.64-.07-.94l2.03-1.58..."/>
  </symbol>

  <symbol id="icon-arrow" viewBox="0 0 24 24">
    <path d="M12 4l-1.41 1.41L16.17 11H4v2h12.17l-5.58 5.59L12 20l8-8z"/>
  </symbol>
</svg>
```

## Usage with Buttons

```tsx
import { Icon, Button } from "@shalecss/react";

const IconButton = () => {
  return (
    <Button variant="primary">
      <Icon icon="save" width={16} height={16} />
      Save
    </Button>
  );
};
```

## Styling Icons

You can apply additional styles and classes:

```tsx
<Icon
  icon="heart"
  className="custom-icon"
  style={{ color: 'red' }}
  width={32}
  height={32}
/>
```

## Accessibility

Add aria labels for accessibility:

```tsx
<Icon
  icon="search"
  aria-label="Search"
  role="img"
/>
```

## Notes

- The `icon` prop should match the ID in your SVG sprite (without the "icon-" prefix)
- Icons inherit the current text color by default
- Use `width` and `height` props from SVGAttributes to control size
- The component uses SVG's `<use>` element for efficient icon rendering
