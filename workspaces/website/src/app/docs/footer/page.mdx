import { Footer } from "@shalecss/react";

The `Footer` component provides a styled footer container for page footers.

## Props

- `Component` (optional): The underlying HTML element to render. Defaults to `footer`.

## Example

<Footer>
  <p>&copy; 2024 My Company. All rights reserved.</p>
</Footer>

```tsx
import { Footer } from "@shalecss/react";

const MyComponent = () => {
  return (
    <Footer>
      <p>&copy; 2024 My Company. All rights reserved.</p>
    </Footer>
  );
};
```

## Complete Footer Example

```tsx
import { Footer, Container, FlexContainer, Link } from "@shalecss/react";

const PageFooter = () => {
  return (
    <Footer>
      <Container>
        <FlexContainer variant="space-between">
          <div>
            <p>&copy; 2024 My Company</p>
          </div>
          <nav>
            <Link href="/privacy">Privacy Policy</Link>
            <Link href="/terms">Terms of Service</Link>
            <Link href="/contact">Contact</Link>
          </nav>
        </FlexContainer>
      </Container>
    </Footer>
  );
};
```

## Polymorphic Usage

The Footer component is polymorphic and can be rendered as any HTML element:

```tsx
<Footer Component="div">
  Footer content as a div
</Footer>
```
