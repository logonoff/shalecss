import { SkipToContent } from "@shalecss/react";

The `SkipToContent` component provides an accessible "skip to main content" link that helps keyboard and screen reader users bypass navigation and jump directly to the main content.

## Props

- `Component` (optional): The underlying HTML element to render. Defaults to `a`.

## Example

### Basic Usage

<SkipToContent href="#main-content">
  Skip to main content
</SkipToContent>

```tsx
import { SkipToContent } from "@shalecss/react";

const MyLayout = ({ children }) => {
  return (
    <>
      <SkipToContent href="#main-content">
        Skip to main content
      </SkipToContent>

      <nav>{/* Navigation items */}</nav>

      <main id="main-content">
        {children}
      </main>
    </>
  );
};
```

## Complete Accessibility Example

```tsx
import { SkipToContent, Nav, MenuBarButton } from "@shalecss/react";

const AccessibleLayout = ({ children }) => {
  return (
    <div>
      {/* Skip link - visually hidden until focused */}
      <SkipToContent href="#main">
        Skip to main content
      </SkipToContent>

      {/* Navigation */}
      <Nav>
        <MenuBarButton href="/">Home</MenuBarButton>
        <MenuBarButton href="/about">About</MenuBarButton>
        <MenuBarButton href="/docs">Docs</MenuBarButton>
      </Nav>

      {/* Main content with matching ID */}
      <main id="main" tabIndex={-1}>
        {children}
      </main>
    </div>
  );
};
```

## Multiple Skip Links

You can provide multiple skip links for complex layouts:

```tsx
<div>
  <SkipToContent href="#main-content">
    Skip to main content
  </SkipToContent>
  <SkipToContent href="#nav">
    Skip to navigation
  </SkipToContent>
  <SkipToContent href="#footer">
    Skip to footer
  </SkipToContent>

  <nav id="nav">{/* Navigation */}</nav>
  <main id="main-content">{/* Content */}</main>
  <footer id="footer">{/* Footer */}</footer>
</div>
```

## Polymorphic Usage

```tsx
import { Link } from "react-router-dom";

<SkipToContent Component={Link} to="#main">
  Skip to main content
</SkipToContent>
```

## Behavior

The SkipToContent component:
- Is visually hidden by default
- Becomes visible when focused (via keyboard navigation)
- Allows users to bypass repetitive navigation
- Improves accessibility for keyboard and screen reader users

## Best Practices

1. **Place at the beginning** - The skip link should be the first focusable element on the page
2. **Use descriptive text** - "Skip to main content" is clear and standard
3. **Match IDs** - Ensure the `href` matches the `id` of the target element
4. **Set tabIndex** - Consider setting `tabIndex={-1}` on the target element to ensure it receives focus

```tsx
<main id="main-content" tabIndex={-1}>
  {/* Content */}
</main>
```

## WCAG Compliance

This component helps meet WCAG 2.1 Success Criterion 2.4.1 (Bypass Blocks), which requires a mechanism to bypass blocks of content that are repeated on multiple pages.
