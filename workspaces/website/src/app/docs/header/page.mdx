import { Header, HeaderTitle, HeaderText } from "@shalecss/react";

Header components provide structured page header layouts with title and text elements.

## Header Component

The main header container component.

### Props

- `state` (optional): The initial state of the header. Can be one of:
  - `default`: The default state
  - `hover`: The hover state
  - `active`: The active state
  - `disabled`: The disabled state
- `Component` (optional): The underlying HTML element to render. Defaults to `header`.

### Example

<Header>
  <HeaderTitle>Page Title</HeaderTitle>
  <HeaderText>Subtitle or description</HeaderText>
</Header>

```tsx
import { Header, HeaderTitle, HeaderText } from "@shalecss/react";

const MyComponent = () => {
  return (
    <Header>
      <HeaderTitle>Page Title</HeaderTitle>
      <HeaderText>Subtitle or description</HeaderText>
    </Header>
  );
};
```

### With State

```tsx
<Header state="active">
  <HeaderTitle>Active Header</HeaderTitle>
</Header>
```

## HeaderTitle Component

A component for header titles.

### Props

- `Component` (optional): The underlying HTML element to render. Defaults to `div`.

### Example

```tsx
<HeaderTitle>My Application</HeaderTitle>
```

## HeaderText Component

A component for header text/descriptions.

### Props

- `Component` (optional): The underlying HTML element to render. Defaults to `p`.

### Example

```tsx
<HeaderText>Welcome to our documentation</HeaderText>
```

## Complete Example

```tsx
import { Header, HeaderTitle, HeaderText } from "@shalecss/react";

const PageHeader = () => {
  return (
    <Header>
      <HeaderTitle>Documentation</HeaderTitle>
      <HeaderText>
        Learn how to use our component library
      </HeaderText>
    </Header>
  );
};
```

## Polymorphic Usage

All header components are polymorphic:

```tsx
<Header Component="div">
  <HeaderTitle Component="h1">Main Title</HeaderTitle>
  <HeaderText Component="span">Description text</HeaderText>
</Header>
```
