import { CaptionButton, CaptionMenu } from "@shalecss/react";

Caption button components provide specialized button and menu styling, typically used for window controls or compact UI elements.

## CaptionButton Component

A polymorphic button component with caption styling.

### Props

- `Component` (optional): The underlying HTML element to render. Defaults to `button`.

### Example

<CaptionButton>×</CaptionButton>

```tsx
import { CaptionButton } from "@shalecss/react";

const MyComponent = () => {
  return (
    <CaptionButton onClick={handleClose}>
      ×
    </CaptionButton>
  );
};
```

### Window Controls Example

```tsx
<div style={{ display: 'flex', gap: '8px' }}>
  <CaptionButton onClick={handleMinimize}>
    −
  </CaptionButton>
  <CaptionButton onClick={handleMaximize}>
    □
  </CaptionButton>
  <CaptionButton onClick={handleClose}>
    ×
  </CaptionButton>
</div>
```

## CaptionMenu Component

A companion component for caption menu styling.

### Props

- `Component` (optional): The underlying HTML element to render. Defaults to `span`.

### Example

<CaptionMenu>☰</CaptionMenu>

```tsx
import { CaptionMenu } from "@shalecss/react";

const MyComponent = () => {
  return (
    <CaptionMenu>
      ☰
    </CaptionMenu>
  );
};
```

## Combined Usage

```tsx
import { CaptionButton, CaptionMenu } from "@shalecss/react";

const WindowControls = () => {
  return (
    <div className="window-controls">
      <CaptionMenu>☰</CaptionMenu>
      <CaptionButton onClick={handleMinimize}>−</CaptionButton>
      <CaptionButton onClick={handleMaximize}>□</CaptionButton>
      <CaptionButton onClick={handleClose}>×</CaptionButton>
    </div>
  );
};
```

## Polymorphic Usage

Both components are polymorphic:

```tsx
<CaptionButton Component="div" onClick={handleClick}>
  Custom element
</CaptionButton>
```
