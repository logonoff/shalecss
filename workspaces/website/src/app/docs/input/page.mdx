import { Input } from "@shalecss/react";

The `Input` component is a wrapper around the HTML `<input>` element with built-in label support and styling.

## Props

*extends `React.InputHTMLAttributes<HTMLInputElement>`*

- `label` (optional): The label text for the input. When provided with an `id`, it will render an associated label element.
- `id` (optional): The id for the input. Used to associate the input with its label.

## Example

### Basic Input

<Input type="text" placeholder="Enter text..." />

```tsx
import { Input } from "@shalecss/react";

const MyComponent = () => {
  return <Input type="text" placeholder="Enter text..." />;
};
```

### Input with Label

<Input id="username" label="Username" type="text" />

```tsx
import { Input } from "@shalecss/react";

const MyComponent = () => {
  return (
    <Input
      id="username"
      label="Username"
      type="text"
    />
  );
};
```

## Notes

When both `label` and `id` are provided, the component automatically creates an accessible label-input pair using the `htmlFor` attribute.
