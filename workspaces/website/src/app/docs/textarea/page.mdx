import { Textarea } from "@shalecss/react";

The `Textarea` component is a wrapper around the HTML `<textarea>` element with consistent styling.

## Props

*extends `React.TextareaHTMLAttributes<HTMLTextAreaElement>`*

This component accepts all standard HTML textarea attributes like `rows`, `cols`, `placeholder`, etc.

## Example

<Textarea placeholder="Enter your message..." rows={4} />

```tsx
import { Textarea } from "@shalecss/react";

const MyComponent = () => {
  return (
    <Textarea
      placeholder="Enter your message..."
      rows={4}
    />
  );
};
```

## With Custom Styling

```tsx
<Textarea
  className="custom-class"
  placeholder="Type here..."
  rows={6}
  defaultValue="Initial content"
/>
```
