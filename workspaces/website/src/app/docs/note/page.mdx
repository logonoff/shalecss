import { Note, NoteText } from "@shalecss/react";

The `Note` component displays contextual information with different visual styles based on the type of message.

## Note Component

### Props

*extends `FlexItem` props*

- `variant` (required): The type of note. Can be one of:
  - `info`: Informational messages
  - `alert`: Alert or error messages
  - `warn`: Warning messages
  - `tip`: Helpful tips
- `Component` (optional): The underlying component to render. Defaults to `FlexItem`.

### Example

<Note variant="info">
  <NoteText>This is an informational note.</NoteText>
</Note>

<Note variant="alert">
  <NoteText>This is an alert message.</NoteText>
</Note>

<Note variant="warn">
  <NoteText>This is a warning message.</NoteText>
</Note>

<Note variant="tip">
  <NoteText>This is a helpful tip.</NoteText>
</Note>

```tsx
import { Note, NoteText } from "@shalecss/react";

const MyComponent = () => {
  return (
    <>
      <Note variant="info">
        <NoteText>This is an informational note.</NoteText>
      </Note>

      <Note variant="alert">
        <NoteText>This is an alert message.</NoteText>
      </Note>

      <Note variant="warn">
        <NoteText>This is a warning message.</NoteText>
      </Note>

      <Note variant="tip">
        <NoteText>This is a helpful tip.</NoteText>
      </Note>
    </>
  );
};
```

## NoteText Component

A companion component for styling text inside notes.

### Props

- `Component` (optional): The underlying HTML element to render. Defaults to `p`.

### Example

```tsx
<Note variant="info">
  <NoteText>Important: Read the documentation carefully.</NoteText>
</Note>
```

## Polymorphic Usage

Both components are polymorphic:

```tsx
<Note variant="tip" Component="section">
  <NoteText Component="span">Pro tip: Use keyboard shortcuts!</NoteText>
</Note>
```
